# SAI2L Recovery

SAI2L Recovery is a local tool designed to help users who have legitimately purchased Paint Tool SAI 2 but have lost access to their official license key. It operates entirely offline in your browser to patch the executable and generate a recovery license file.

## Features
- **Offline Processing:** All patching and key generation is done locally in your browser using JavaScript. No files are uploaded to any server.
- **Multilingual Support:** English, Simplified Chinese, Traditional Chinese, and Korean.
- **Automated Bundling:** Automatically patches the `.exe`, generates the `.slc` file, and bundles them into a ZIP archive for easy extraction.

## Usage
1. Locate your Paint Tool SAI 2 installation folder.
2. Open `index.html` in your web browser.
3. Upload your original `sai2.exe` file.
4. Input your **System ID** (found by opening your unlicensed SAI2 app -> `Other` -> `System ID`).
5. Click **Fix & Download ZIP**.
6. Extract the downloaded ZIP file and replace the original files in your SAI2 folder.
